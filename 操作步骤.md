# 创建项目

### 从头创建项目

1. 创建一个新的 Next.js 项目：
```
npx create-next-app threejs-geojson-map
```

在安装过程中，选择以下选项：

- 使用 TypeScript: 是
- 使用 ESLint: 是
- 使用 Tailwind CSS: 是
- 使用 `src/` 目录: 否
- 使用 App Router: 是
- 自定义导入别名: 是 (使用默认的 `@/*`)
